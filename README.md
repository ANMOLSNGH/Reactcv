# 📄 Interactive CV Builder

A modern, clean, and interactive web application designed to help users build and preview professional resumes in real-time. Built with a focus on elegant typography, crisp visual hierarchy, and a smooth user experience.

## ✨ Features

- **Real-Time Preview:** Watch your CV update instantly as you edit your details.
- **Modern UI/UX:** Styled with a sleek, minimalist "SaaS-style" dashboard layout, balanced padding, and smooth micro-interactions.
- **Responsive Design:** Optimized for a flawless viewing experience across various screen sizes.
- **Production Ready:** Bundled and optimized using Parcel for blazing-fast performance.

## 🛠️ Tech Stack

- **Frontend:** HTML5, Modern CSS3 (featuring custom CSS custom properties and flexbox layouts)
- **Scripting:** JavaScript (ES6+)
- **Bundler & Build Tool:** Parcel

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git](https://github.com/YOUR_USERNAME/YOUR_REPOSITORY_NAME.git)
   cd YOUR_REPOSITORY_NAME
2. npm install
3. npx parcel index.html
4. Building for Production
To create an optimized, production-ready build in the dist folder:
npx parcel build index.html
